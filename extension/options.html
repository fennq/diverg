Diverg Extension Options

Web scan API

If set, full scans use this URL. If empty, the extension auto-detects http://127.0.0.1:5000 or http://localhost:5000 when you run the API locally.

Auto-scan on visit is in the popup under Options ▾. It runs in the browser (headers, scripts, forms) and shows results in the side panel.

Solana (side panel)

Required for the Solana tab → Analyze bundle. The key is stored only in this browser and sent directly to Helius. Get a key.

Chain investigation (optional)

Paste the same token the web console stores after login (diverg_token in localStorage). Used only from this browser for Full API in the popup address harvester. On the server, set DIVERG_ALLOW_EXTENSION_CORS=1 so Chrome can call your API from the extension origin.
