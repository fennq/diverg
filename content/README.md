# Content

Documentation, thread templates, and diagram assets for reports and social.

- **okara-scan-thread.md** — Thread copy and steps to run a scan and screenshot diagram cards.
- **okara-scan-diagrams.html** — Screenshot-ready diagram cards (severity breakdown, what we checked, safe badge). Open in browser; use URL params for counts.
- **scripts/scan_url.py** (repo root) — One-off full scan from CLI; output JSON for use in threads or docs.

Scan result JSON files in `content/` are gitignored; do not commit them.
