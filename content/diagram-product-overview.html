Diverg — Product Overview
Adaptive security testing that discovers first, then runs only what fits
INPUT
Target
URL or domain (e.g. example.com, https://app.example.com). Authorized scope only.
HOW IT WORKS
1. Discover
Recon, headers, techstack. Map subdomains, APIs, frameworks.
→
2. Adapt
Infer profiles (crypto-trading, API, admin…). Pick skills that match.
→
3. Run
Only relevant tools: payment, IDOR, client-side intel, CVE, logic abuse.
→
4. Correlate
Rank findings, attack paths, recommended next tests.
→
5. Report
LLM summary, evidence-backed, CONFIRMED vs POSSIBLE.
Output: Findings + fixes, attack path hypotheses, follow-up recommendations. No sensitive data exposed in reports.
