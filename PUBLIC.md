# What’s public in this repo

We keep the public face of Diverg minimal. This file describes what we intentionally share.

## Public

- **README.md** — High-level description of Diverg and how to follow progress.
- **RECENT_UPDATES.md** — Day-to-day updates so we can show we’re actively working on the product.
- **Chrome extension** — The free public extension lives in its own repo: [diverg-extension](https://github.com/fennq/diverg-extension). It does **not** perform blockchain, wallet, or crypto-specific scanning.

## Internal / private

- Full codebase (orchestrator, skills, API server, runbooks, exploit catalog, internal docs) is for internal use and authorized contributors only. We don’t document or advertise internals in the public README.

If you’re contributing or need access to more than the above, reach out to the team.
